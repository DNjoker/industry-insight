# 信息汇总桌面工具 — 功能说明

## 概述

一个桌面端电商运营工具，基于 AI 联网搜索 + 大语言模型，覆盖行业分析、卖点生成、竞品研究、知识管理四大场景。所有产出自动保存到 Obsidian Vault，支持知识库语义检索。

**技术栈**：Electron + React（前端）/ Python FastAPI（后端）/ DeepSeek（LLM）/ ChromaDB（向量库）

---

## 一、行业摸底

**做什么**：输入一个品类/赛道，AI 自动联网搜索并生成结构化的行业分析报告。

**输入**：
- 行业/品类名称（如"宠物免洗手套"）
- 时间范围：近一周 / 近一月 / 近半年 / 不限

**输出**（自动保存到 Obsidian `行业摸底/` 目录）：
- 一份 Markdown 报告，包含：摘要、价值链、竞争格局、主要玩家、消费者行为与痛点、经营打法、品牌格局、渠道玩法、爆品与趋势、竞品拆解、素材与定价分析
- 一份运营行动清单（具体的下一步建议）
- 所有网络搜索原始来源文章

**特色**：
- SSE 流式生成，实时看到搜索 → 抓取 → 分析 → 保存四个阶段进度
- 支持热门行业推荐和自动补全
- 报告可直接关联到"卖点整理"模块，AI 生成文案时会吸收报告内容

**使用建议**：进新品类前先跑一遍，5-10 分钟内能拿到一份有深度的摸底报告。

## 二、卖点整理

**做什么**：根据产品信息和目标人群，为各电商/内容平台生成主图、详情页、种草文案、视频脚本。

**两种模式**：

### 电商平台模式
- 支持平台：淘宝、拼多多、抖音、京东
- 可多选，生成后可对比不同平台的文案差异
- 每个平台输出：主图文案（3-5条）、FAB 卖点（3-5组）、竞品差评挖掘、详情页排版（6-8屏）、主图设计简报（配色/字体/构图/画面方向）、附图设计、跨平台差异说明、投放关键词

### 内容平台模式
- 支持平台：小红书、公众号、抖音内容、快手、视频号、B站
- 单选，每个平台独立生成
- 输出内容：
  - **小红书**：图文种草（标题选项、正文框架、标签策略、配图方案、SEO建议）
  - **公众号**：长文结构（标题、导语、正文流、观点、金句、CTA）
  - **抖音内容/快手/视频号**：短视频脚本（钩子方向、脚本大纲、分镜表含景别/运镜/口播/BGM/字幕）
  - **B站**：中长视频脚本（含弹幕互动点和信息深度设计）

**输入字段**：
| 字段 | 说明 |
|------|------|
| 产品名称 | 必填 |
| 核心卖点/成分 | 必填，越详细越好 |
| 所属类目 | 影响行业模板匹配 |
| 目标人群 | 如"25-35岁一线城市女性白领"，决定视觉配色和风格 |
| 价格区间 | 如"39-59元" |
| 价格定位 | 高端 / 中端 / 性价比，决定沟通策略和视觉方向 |
| 用户痛点 | 可选，不填则 AI 自行推导 |
| 产品优势/劣势 | 可选，AI 会放大优势、转化劣势 |
| 竞品参考 | 可选，或从灵感库导入 |
| 品牌调性 | 可选，如"年轻潮酷""专业可信""温馨居家" |
| 关联行业报告 | 可选，从行业摸底选择报告文件 |

**特色功能**：

1. **自检+优化**：生成后 AI 自动打分（0-100）并列出问题清单。点击"优化"按问题逐项修复，支持多轮优化。每轮自动保存历史，可回滚到任意之前版本。

2. **对比模式**（电商）：同时展示所有选中平台的文案，横向对比差异。

3. **7 个行业模板**：通用 / 宠物用品 / 美妆护肤 / 食品饮料 / 3C数码 / 服装鞋包 / 母婴用品。每个模板内置了行业特定的示例、FAB 侧重和详情页节奏。

4. **用户自定义模板**：可导入/导出 JSON 模板文件。

5. **受众×价格视觉矩阵**：AI 根据目标人群和价格定位，从 9 类人群 × 3 个价格带中匹配最合适的视觉方案（配色/配图/字体/排版）。

6. **提示词编辑器**：高级用户可自由编辑完整 system prompt。

7. **局部重新生成**：只重做某个平台的某个部分（如"拼多多的详情页"），不浪费 token。

8. **自动保存+恢复**：刷新页面不丢数据，两个模式各自独立保存。

**使用建议**：
- 电商模式先选 1 个平台生成，满意后再加其他平台对比
- 第一轮评分通常在 60-75 分，优化 1-2 轮可到 80-88 分
- 90 分以上极难达到——AI 无法真正判断"这个钩子能不能打动人"
- 内容平台每次只做一个，专注度更高

## 三、竞品分析

**做什么**：内嵌浏览器打开竞品商品页，截图后用视觉 AI 分析页面设计和文案。

**支持平台**：淘宝、拼多多、抖音、京东

**流程**：
1. 在内嵌浏览器中输入竞品商品链接
2. 点击"捕获页面"自动抓取页面截图
3. 选择要分析的截图（可多选）
4. AI 视觉模型分析每张截图：文案、布局、配色、视觉焦点、卖点
5. 综合评估：结构/文案/视觉质量评分 + 可借鉴思路 + 可改进点

**评论分析**：自动抓取商品评论，AI 分析差评/好评主题、竞争差距、复购信号、价格感知。

**灵感库**：
- 所有分析结果 + 截图保存到 Obsidian `灵感库/` 目录
- 可以浏览、搜索、删除
- 支持导入到卖点整理作为竞品参考

**使用建议**：竞品分析 + 评论抓取双管齐下，比只看页面更全面。

## 四、知识榨取

**做什么**：批量粘贴文章链接或笔记，AI 自动提炼为结构化知识卡片。

**流程**：
1. 粘贴链接/文本（一行一条，可加备注用 `|` 分隔）
2. AI 自动识别 URL 还是纯文本
3. 抓取网页内容 → AI 提炼标题、标签、摘要、正文
4. 人工审核环节：可编辑/删除卡片，补充想法
5. 保存到 Obsidian `知识卡片/` 目录

**特色**：支持长文保留原文和 AI 提炼两种模式，自动判断。

**使用建议**：日常看到好文章、竞品动态、行业报告就扔进去，积累自己的知识库。后续在策略对谈中可以通过知识库检索调用。

---

## 五、策略对谈

**做什么**：带有知识库检索和网络搜索能力的 AI 策略顾问。

**三种模式**：
- **对话模式**：简洁的苏格拉底式问答，帮你理清思路
- **详细模式**：结构化长篇分析，适合深度策略讨论
- **闲聊模式**：随意对话，无电商预设

**特色**：
- 可以开关知识库检索（搜索你之前积累的知识卡片和行业报告）
- 可以开关网络搜索（获取最新信息）
- 自动提取关键词提升搜索精度
- 对话可保存到 Obsidian
- 当知识库缺少某个主题时，会自动建议"去行业摸底"

**使用建议**：先做行业摸底和知识榨取积累素材，再来对谈会更精准。

---

## 六、对话梳理

**做什么**：导入 DeepSeek 聊天记录，AI 提炼核心观点。

**两种输入方式**：
- 直接粘贴聊天文本
- 从 Obsidian 加载已保存的对话文件

**输出**：
- 核心想法、关键信息、共识、待办事项
- 保存到 Obsidian `DeepSeek对话/已提炼/` 目录

**浏览器扩展**：配套 Chrome 插件可一键导出 DeepSeek 聊天内容到 Obsidian。

**使用建议**：和 AI 聊完策略后导出，避免重要结论被淹没在长对话里。

## 七、配置

| 配置项 | 说明 |
|--------|------|
| AI 模型 | DeepSeek / Claude / OpenAI，可自定义模型名和 Base URL |
| 搜索引擎 | Tavily（推荐，每月 1000 次免费）/ Bing API / 直接抓取 |
| Obsidian Vault | 本地 Obsidian 仓库路径，所有内容保存到此 |
| 视觉模型 | 竞品分析用，支持豆包/通义千问/GPT-4o |
| 知识库 | ChromaDB 同步设置，可自定义同步目录和集合名 |

---

## 常见工作流

**新品类切入**：行业摸底 → 查看报告 → 一键关联到卖点整理 → 生成文案

**竞品监控**：竞品分析 → 截图+评论分析 → 保存到灵感库 → 导入卖点作为参考

**日常学习**：看到好文章 → 知识榨取 → 存入知识库 → 策略对谈中检索复用

**策略推演**：策略对谈 → 深度讨论 → 保存对话 → 对话梳理提炼要点

---

## 注意事项

1. 首次使用需在设置页配置 AI API Key 和 Obsidian Vault 路径
2. 行业摸底和知识榨取需要网络搜索功能（推荐用 Tavily 免费额度）
3. 知识库功能需要安装 ChromaDB 依赖
4. 竞品分析的内嵌浏览器功能需要在 Electron 桌面环境中使用
5. 浏览器扩展仅支持 Chrome / Edge（Chromium 内核）
